# payment sdk dashboard

Provides a Web Dashboard for the Payment SDK

## Requirements

- node (10+ recommended)
- Docker

## Installation requirements on macOS

```bash
brew install node
brew install docker
```

## Development

```bash
git clone git@github.com:mobilabsolutions/payment-sdk-dashboard-open
npm install
npm run dev
```

## Docker

To run it docker

```bash
npm run docker
```
